一.列表
  1.列表的作用
    将具有相似特征或具有先后顺序的内容按照从上到下（从左到右）的顺序排列。
  2.列表分类
    有序列表：<ol></ol>---order list
    无序列表：<ul></ul>---unorder list
  3.列表项
    列表项用于显示数据。
    <li></li>---list item

    ex:
    <ol>
      <li>列表项1</li>
      <li>列表项2</li>
      ...
    </ol>
  4.属性
     1.有序列表属性：
       (1)type:有序列表的排序类型
	  取值：
	   1 数字排序，默认值
	   a 小写字母排序
	   A 大写字母排序
	   I 大写罗马数字 
	   i 小写罗马数字

       (2)start:从第 几 个位置开始排序
	  取值：无单位的数字
     2.无序列表属性：
       (1)type:定义无序列表的标识
          取值：
	    disc 实心圆点
	    circle 空心圆
	    square 实心方块
	    none 无标识
     3.列表嵌套
       在一个列表中去嵌套另外一个列表，永远只能在li中去嵌套。
       语法：
       <ul>
         <li>
	   数据
	   <ol>
	     <li>数据</li>
	   <ol>
	 </li>
       </ul>
二.CSS概述
  1.什么是CSS？
    CSS：Cascading Style Sheets级联样式表，层叠样式表，简称样式表。
  2.CSS的作用
    定义HTML页面中元素的样式(修饰元素)。
  3.CSS样式优点
    1.实现内容与表现的分离，便于代码的阅读和维护
    2.实现代码的重用性和可维护性
  4.使用原则
    w3c规定尽量使用css的属性去取代HTML的属性来修饰元素。
三.CSS的使用方式
   1.内联样式
     也称为行内样式
     将样式定义在html元素的style属性中
     语法：
       <any style="样式声明"></any>
       样式声明：由多个样式“属性”和“值”组成，属性和值之间用冒号连接。
       ex:
       <div style="color:red"></div>
       常用的属性和值：
          定义文字颜色color，取值为合法的颜色值（英文单词）
	  定义背景颜色background,取值颜色的英文单词
	  定义字体大小font-size,取值以px为单位的数字

   2.内部样式
     将样式定义在html头<head></head>元素中
     在head元素中添加一对<style></style>标记，在style标记中添加样式规则。
     语法：
        <head>
	  <style>
	    样式规则
	  </style>
	</head>
	样式规则：由选择器和样式声明组成。
	选择器：规范页面中哪些元素能够使用定义好的样式。
	元素选择器：由标签名称作为选择器，规范当前标记的样式。
	选择器{样式声明}
	ex:
	   div{color:red;}

   3.外部样式
     将样式定义在外部的css文件中，在网页中引入这个css文件。
     特点：样式规则保存在以.css为后缀的文件中
     使用步骤：
        1.创建一个独立的*.css文件
	2.在css文件中添加样式规则
	3.在网页的头部引入css文件
	  <link rel="stylesheet" href="css文件的路径">

     练习：新建一个网页，在网页中添加一个div元素和一个h2元素，内容随意。
        1.用内部样式设置div的字体大小为30px，文本颜色为红色，背景颜色为蓝色。
	2.用外部样式设置h2元素中的文本颜色为黄色，背景颜色灰色。
四.CSS的样式特征
   1.继承性
     大部分的样式属性可以被继承（子元素继承父元素的样式）
     ex:
     div{
	   color:blue;
	   font-size:32px;
	 }

     <div>
	  这是div元素
	  <p>
	    这是嵌套在div中的p元素
	     <span>这是嵌套在div中的p元素中的span元素</span>
	  </p>
	</div>
   2.层叠性
     为一个元素定义多个样式，当样式声明不冲突时，多个样式叠加到一起被元素应用。
     ex:
      div{
	   color:blue;
	  
	 }
      div{
	   font-size:32px;
	 }
      <div style="background:gray;">测试文本</div>

   3.优先级
     如果样式冲突时，按照样式规则的优先级来应用样式
     默认优先级的排序：
        浏览器的默认设置    低
	内部样式和外部样式  居中（就近原则）
	内联样式            高

   4.调整显示的优先级
     使用!important规则调整样式的优先级。
     将!important放在样式属性值之后，与值之间用空格隔开即可。
     ex:
       color:red !important;
     
     练习：
       1.页面中添加div元素，p元素和h3元素，内容随意。
       2.用内联样式设置div元素的样式为红色字体，蓝色背景，运行页面查看效果。
       3.用内部样式设置p元素的样式为30px,黄色字体，红色背景，运行页面查看效果。
       4.用外部样式设置h3元素的样式为绿色字体，灰色背景，并且在外部样式表中设置p元素的字体大小为60px;引入外部样式表后，查看页面效果，然后再改变内部样式与外部样式的位置，再查看页面效果（尝试用!important改变现实效果）。
